## 📚 **기술 스택**

- **JDK 17**  
- **Spring Boot 3.2.4**  
- **Spring Boot JPA**  
- **Spring Boot Validation**  
- **Spring Boot Security**  
- **Stomp**  
- **QueryDSL**  
- **JWT (JSON Web Token)**  
- **S3, EC2, RDS (AWS)**  
- **MySQL, Redis**  
- **Github Actions, Docker-compose**
